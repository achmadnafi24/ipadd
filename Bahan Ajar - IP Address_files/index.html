Apa itu IP Address?

IP Address (Internet Protocol Address) adalah alamat unik yang diberikan kepada setiap perangkat yang terhubung ke jaringan komputer. IP Address berfungsi sebagai identitas perangkat agar dapat saling berkomunikasi.

Contoh IPv4:

192.168.1.1

IPv4 terdiri dari 32 bit yang dibagi menjadi 4 oktet, masing-masing 8 bit. Setiap oktet dipisahkan dengan tanda titik dan dinyatakan dalam bilangan desimal (0–255).

IP Address terbagi menjadi dua bagian: Network ID (identitas jaringan) dan Host ID (identitas perangkat dalam jaringan tersebut).

Kelas IP Address

Kelas A (1.0.0.0 – 126.255.255.255)

Subnet mask default: 255.0.0.0 (/8). Mendukung hingga 16 juta host per jaringan. Digunakan untuk jaringan sangat besar.

Kelas B (128.0.0.0 – 191.255.255.255)

Subnet mask default: 255.255.0.0 (/16). Mendukung hingga 65.534 host per jaringan. Cocok untuk jaringan menengah.

Kelas C (192.0.0.0 – 223.255.255.255)

Subnet mask default: 255.255.255.0 (/24). Mendukung hingga 254 host per jaringan. Paling umum digunakan.

Kelas D & E (224.0.0.0 – 255.255.255.255)

Kelas D untuk multicast, Kelas E untuk eksperimen. Keduanya tidak digunakan untuk pengalamatan host biasa.

Subnetting

Subnetting adalah teknik membagi jaringan besar menjadi beberapa sub-jaringan (subnet) yang lebih kecil. Gunakan kalkulator di bawah untuk berlatih menghitung Network Address, Broadcast, dan jumlah host.
